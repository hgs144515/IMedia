Eco-Fest About Page

Hello, and welcome to Eco-Fest!

We are an environmentally friendly group who looks to help the environment by doing fundraisers and events. Eco-Fest is a family friendly festival we will be having only on the third saturday of July. We will be located in a large field in leeds, so make sure you don't miss out! Ticket information can be found here!

All our profits go towards helping the environment through anyway we can, if that might be donating to charities, paying to have plastic removed from the ocean or just cleaning up a field. We help the environment through many ways, but the main way we can help is to show others how they can help save the environment

We teach families how they can help by recycling and not throwing away all their rubbish. we also tell them how big of an effect it can have if they all throw in a helping hand.

If you want to learn more about recycling, why not watch this helpful video!

We also aim to improve current eco systems for the wildlife living there!
